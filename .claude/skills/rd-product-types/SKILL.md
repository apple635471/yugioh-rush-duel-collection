# Rush Duel 產品類型辨識

從 set ID 前綴辨識產品類型:

| 前綴 | 類型 | 說明 |
|------|------|------|
| `KP` | booster | 補充包 (Booster Pack) — 主要產品線，KP01~KP23+ |
| `ST` | starter | 預組 (Starter Deck) |
| `CP` | character_pack | 角色包 (Character Pack) |
| `GRC` | go_rush_character | Go Rush 角色包 |
| `GRD` | go_rush_deck | Go Rush 預組 |
| `B0xx` | battle_pack | 戰鬥包 (Battle Pack) — 第一期 |
| `B2xx` | battle_pack | 戰鬥包 (Battle Pack) — 第二期 |
| `MAX` | maximum_pack | Maximum 包 |
| `EXT` | extra_pack | Extra 包 |
| `LGP` | legend_pack | 傳說包 (Legend Pack) |
| `SD` | structure_deck | 構築包 (Structure Deck) |
| `VSP` | vs_pack | VS 包 |
| `TB` | tournament_pack | 大會包 (Tournament Pack) |
| `AP` | advanced_pack | 進階包 (Advanced Pack) |
| `ORP` | over_rush_pack | Over Rush 包 |
